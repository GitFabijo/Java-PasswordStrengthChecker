# PasswordStrengthChecker (Java)

Ein kleines Konsolenprogramm in Java, das überpüft. ob dein Passwort lang genug ist und ob es mindestens eine Zahl enthählt. 

## Features 
- Abfrage des Passwortes über Konsole
- validiert die Passwortlänge (mindesrens 8 Zeichen)
- Gibt Ergebnisse als boolean zurück

## Hinweis
Dieses Projekt wurde als Lernübung umgesetzt.
Fokus: Vertändnis von Schleifen, boolean, scanner und Methoden.
